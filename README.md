# アプリケーション名

ROAD BICYCLE（ロード バイシクル）

このアプリケーションは、自転車に乗ることが好きな人達が普段のサイクリングコースを共有したり、一緒に走る日時や場所の連絡ツールとして活用するなどのツールとして、また競技のために日々鍛えている人の自己管理のために日々のトレーニングメニュー等を記録できるようにするために作成したものです。

# アプリケーション概要

① 皆さんの投稿した記事（コース紹介、一緒に走る人を探す、その他自転車に関すること）を閲覧することができます。

② ログインしているユーザーは日々の予定、トレーニングメニュー、走行距離を入力できます。

③ ログインしているユーザーは記事を投稿できます。

④ ログインしているユーザーは自分が投稿した記事の編集や削除ができます。

⑤ ログインしているユーザーは記事に対してコメントの送信ができます。

⑥ 検索窓に調べたいワードを入力すると、そのワードが記載されている記事を閲覧できます。

# 目指した課題解決

① 自分が趣味のロードバイクに乗って出かける時に、自分と同じようにロードバイクに乗っている方々を何人も見かけますが、一人で走っている方が多い印象を受けました（もちろん数人で走っているグループも見かけますが）。その時、「何かしら便利な連絡ツールがあると一緒に楽しく走れるのではないか」と思い、このアプリケーションの作成を思いつきました。

② また、レースに出場している競技志向の方にも使っていただきたいと考え、1日単位でのトレーニングメニューの組み立てや走行距離等を管理できる機能も設けました。

| ------------------ | ------ | ------------------------ |

- has_many :posts
